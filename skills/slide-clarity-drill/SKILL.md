# Slide Clarity Drill

Use this skill when a presentation is already structurally sound but some on-slide wording still feels vague, inflated, generic, or under-qualified.

## Goal

Turn fuzzy slide text into wording that is:

- concrete
- defensible
- brief enough for slides
- closer to the speaker's own voice

## Workflow

1. Scan only the active deck text relevant to the current request.
2. Pick one fuzzy line at a time.
3. Quote the current line and explain briefly why it is unclear.
4. Ask one focused clarification question about the intended meaning.
5. Offer 2-4 candidate rewrites only when useful.
6. Get the user's choice or wording.
7. Move to the next line.
8. After enough lines are resolved, patch them together and validate text fit.

## Question Style

- Ask about meaning, not style preferences in the abstract.
- Prefer: `What do you want this to mean most clearly?`
- Use short option lists when they help the user choose quickly.
- Keep the user on one line at a time; do not batch too many unresolved lines unless requested.

## Rewrite Rules

- Prefer plain claims over rhetorical slogans.
- Replace abstract nouns with concrete functions when possible.
- Avoid committee language like `value-add`, `complements`, `bridging`, `supports` unless the user explicitly wants that tone.
- If a rewrite weakens visual hierarchy because it requires smaller type, shorten it again instead of shrinking the font.
- Keep slide wording shorter than note wording; move nuance into notes if needed.

## Good Patterns

- `Research helps me teach principles, not only tools.`
- `This adds a useful specialization within a broad department.`
- `Meetups and the conference keep me grounded in current practice.`

## Bad Patterns

- `This creates synergies across multiple domains.`
- `The durable core of the next web.`
- `Supports outward-facing educational infrastructure.`

## Finish

When the pass is complete:

- patch all agreed wording
- rebuild the deck
- run text validation
- summarize which lines were clarified
